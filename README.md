It is an extension which displays a timer while using some social media platforms and shows a message on every 30 mins of activity.
copy the source code in a file then unload it in chrome browser and when the smaller timer(bottom right) reaches 30 mins you will receive a message if you are using youtube,instagram,tiktok or facebook.
